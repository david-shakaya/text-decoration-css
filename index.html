Свойства CSS с примерами


Свойство "text-decoration"

Задаёт различное оформительное подчёркивание и другие эффекты текста.

  • text-decoration: underline - Создает нижнее подчеркивание

  • text-decoration: none - Cнимает подчеркивание с сылок

  • text-decoration: line-through - Создает перечеркнутый текст

  • text-decoration: overline - Создает подчеркивание над текстом

Свойство "text-transform"

Управляет регистром символов в тексте, например сделать все буквы заглавными или прописными.
В разметке текст всегда набирается согласно правилам языка. После чего, при помощи text-transform, приводится к нужному виду

  • text-transform: uppercase - Стилизирует все буквы в заглавные( например для меню)

  • text-transform: lowercase - Стилизирует все буквы в нижнем регистре

  • text-transform: capitalize - Стилизирует какждую первую букву заглавной

Свойство "text-align"

Управляет выравниванием текстового содержимого элемента по горизонтали.

  • text-align: right -

    Выравнивает текст заголовка и абзаца по правому краю

  • text-align: left - Выравнивает текст заголовка и абзаца по левому краю

  • text-align: center - Выравнивает текст заголовка и абзаца по центру

  • taxt-align: justify - Выравнивает текст равномерно по правому и левому краю( подробнее будет изучаться позже) с
